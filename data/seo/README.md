# data/seo

## Important

- `registry.json` is a generated file.
- Do not edit `registry.json` manually.
- Source of truth is `i18n/pages/*/seo-meta.json`.
- `suffix-config.json` stores per-language common SEO title suffix settings for Patch Console.

## How to regenerate

```bash
npm run seo:registry:generate
```

or

```bash
npm run seo:all:generate
```
