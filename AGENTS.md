# 번역
신규 파일/기능을 만들 때는 항상 /i18n 을 참조하여 각 언어별로 번역이 될 수 있도록 해야합니다.
게임 정보에 대한 사이트이기 때문에 고유 명사가 상당히 많고, 고유 명사를 임의로 의역하지 않고 불분명한 경우가 있는 경우 꼭 사용자에게 재질문 또는 정리를 통해 확인받아주세요.
현재 지원하는 언어는 `en`, `kr`, `jp`입니다. 나중에 `cn`도 추가할 예정이고, 더 많은 언어를 추가할 수도 있어요.

# 서버 특징
en, jp는 kr, cn보다 늦게 출시된 글로벌 서버로 미출시된 콘텐츠들이 많아서 모든 한국어의 모든 내용이 다뤄지지 않는 경우가 많습니다.

# 필수 조건
- 코드 수정하는 과정에서 한국어나 일본어 등 다른 언어를 변환하다가 UTF 에러 등 깨지는 경우가 많은데 항상 이런 일이 없도록 점검할 것
